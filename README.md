## 📂 如何使用

### [📲 示例站点](https://smcfcper.off.gs/)

### 💾 部署本地

1. `git clone git@github.com:FreeDayO/smcfcper.git`或者下载[发行页](https://github.com/FreeDayO/smcfcper/releases)中的源代码包并解压
2. `cd smcfcper`
3. 运行你的本地服务器软件
4. 访问对应页面
